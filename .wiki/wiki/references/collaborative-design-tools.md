# Collaborative Design Tools

> Collaborative design tools help teams externalize ideas as diagrams, whiteboards, prototypes, design systems, and implementation-ready artifacts.

The current sources cover two open-source tools with different centers of gravity. Excalidraw is a hand-drawn-style virtual whiteboard and embeddable React editor. Penpot is a web-based design platform focused on UI design, design systems, code handoff, open standards, self-hosting, and AI workflow integration.

## Tool Comparison

| Tool | Primary role | Notable traits |
|------|--------------|----------------|
| Excalidraw | Whiteboarding and diagrams | Infinite canvas, hand-drawn style, `.excalidraw` JSON format, PNG/SVG export, libraries, localization, hosted app, local-first behavior, real-time collaboration, end-to-end encryption. |
| Penpot | Product/UI design platform | UI design, prototypes, design systems, native tokens, CSS Grid and Flex layouts, open file format, self-hosting, API and integration links, MCP server and AI workflow positioning. |

Excalidraw is strongest when the artifact is a sketch, wireframe, flow, or collaborative diagram. Penpot is stronger when the artifact needs product design structure: components, tokens, responsive layout behavior, and handoff to implementation.
