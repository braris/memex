# Penpot

Source: https://penpot.app/

Fetched: 2026-07-05

## Extracted Content

Penpot is presented as an open-source design platform for teams that need scalable collaboration. The homepage positions it as a full-stack design environment for designers, developers, and AI agents.

The product areas highlighted on the homepage include:

- **UI Design**: responsive and rules-based interface design with wireframes, interactive prototypes, CSS Grid, and Flex layouts.
- **Design Systems**: design components, native design tokens, shared libraries, and visual frameworks intended to scale across teams.
- **AI Workflows**: integration points for agents, LLMs, and AI tooling across code-to-design, design-to-code, and design-to-design workflows.
- **Code**: inspection and integration of code that matches the design, with emphasis on open standards such as CSS, HTML, SVG, JSON, and Penpot's open file format.

The site exposes several product and resource areas from the homepage navigation, including self-hosting, features, release notes, integration and API resources, business cases, plugins, libraries and templates, a learning center, tutorials, a help center, pricing, security and compliance, and enterprise information.

Penpot also advertises MCP-related functionality from its homepage navigation and feature links, including a Penpot MCP Server and AI workflow messaging around multidirectional workflows, MCP server support, and an open API.
