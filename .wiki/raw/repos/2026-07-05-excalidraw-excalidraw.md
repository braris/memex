## README Summary

Excalidraw is an open-source virtual whiteboard with a hand-drawn visual style. The README describes it as a tool for creating diagrams, wireframes, and other sketches. It is collaborative and end-to-end encrypted.

The repository contains both the reusable Excalidraw editor package and the source for the hosted `excalidraw.com` app.

## Editor Package Features

The Excalidraw editor npm package supports:

- Free and open-source usage
- Infinite canvas-based whiteboard
- Hand-drawn-like rendering style
- Dark mode
- Customization
- Image support
- Shape libraries
- Localization and internationalization
- Export to PNG, SVG, and clipboard
- Open `.excalidraw` JSON file format
- Drawing tools such as rectangle, circle, diamond, arrow, line, free draw, and eraser
- Arrow binding and labeled arrows
- Undo and redo
- Zoom and panning

## Hosted App Features

The hosted app at [excalidraw.com](https://excalidraw.com) is described as a minimal showcase of what can be built with Excalidraw. Its source lives in the repository under `excalidraw-app/`.

The README lists these app-level features:

- Progressive web app support, including offline operation
- Real-time collaboration
- End-to-end encryption
- Local-first support with browser autosave
- Shareable readonly links

The README notes that these app-level capabilities are expected to become drop-in plugins for the npm package in the future.

## Quick Start

The README quick start is for integrating the Excalidraw npm package into another application, not for local repository development.

Install with npm:

```bash
npm install react react-dom @excalidraw/excalidraw
```

Or with yarn:

```bash
yarn add react react-dom @excalidraw/excalidraw
```

For local repository development, the README points readers to the Excalidraw development guide.
